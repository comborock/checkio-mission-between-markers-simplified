Dan Vam je string i dva markera (početni i završni). Nađite podstring okružen tim markerima. Ali postoji nekoliko važnih uvjeta.

Ovo je pojednostavljena misija verzije Između markera.

  • Inicijalni i konačni markeri su uvijek različiti.
  • Inicijalni i konačni markeri su uvijek 1 znak (char) veličine.
  • Inicijalni i konačni markeri uvijek postoje u stringu i idu jedno iza drugog.

Ulaz: Tri argumenta. Svaki od njih su tipa string. Drugi i treći arugmenti su početni i konačni (završni) markeri.

Izlaz: String.

Primjer:

{% if interpreter.slug == "js-node" %} 
betweenMarkers('What is >apple<', '>', '<') == 'apple'
{% else %} 
between_markers('What is >apple<', '>', '<') == 'apple'
{% endif %}

Gdje se ovo može primjeniti: Za raščlanjivanje teksta.

Preduvjet: Ne može biti više od jednog konačnog i jednog početnog markera.
